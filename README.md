# React Test Task - Список товаров
Небольшое одностраничное приложение на **React**, тестовое задание.
# Функционал
- Отображение списка товаров (из локального файла `products.js`)
- Поиск товаров по названию на лету
- Модальное окно с подробной информацией о товаре
- Закрытие модалки по:
  - крестику
  - клику вне окна
  - клавише Esc
- Адаптивная верстка под мобильные устройства и планшеты
# Технологии
- **React**
- **JavaScript**
- **CSS**
- **Git**
